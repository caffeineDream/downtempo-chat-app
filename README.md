## What is this?

Node chat application as learning project.

[Heroku deployment link.](https://downtempo-chat.herokuapp.com/)

Client-side: 
 - pure Javascript
 - socket.io

Server-side:
 - Node.js + Express
 - socket.io
 - JWT authentication
